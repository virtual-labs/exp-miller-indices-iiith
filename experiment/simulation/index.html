Instructions
  • This experiment is designed to teach how to visualize the planes in a crystal lattice using miller indices.
  • You will be given a 3D grid with a lattice rendered on the screen. For a given dropdown miller indices, the user should select any three atoms that construct this miller plane.
  • Clicking 'create plane' will create a plane passing through the selected points.
  • Check Miller index will compute the miller index for the constructed plane and will validate with the current miller index of interest in the dropdown just below.
  • Depending on whether the plane constructed has the given miller index then it will results in a 'correct' / 'incorrect' result on the observation section on the right.
